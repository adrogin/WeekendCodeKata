Task:

For a given array with N elements, find a subset of sequential M elements with smallest sum among subsets of the same lenght.

Example

Array of 10 Integer numbers
3 1 6 6 8 2 5 2 9 0

Find the subset of 3 elements with smallest sum

Solution: 2 5 2

Additional requirement: solution must not contain mutable variables. No variable value can be changed after assignment.
